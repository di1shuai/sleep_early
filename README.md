# sleep_early

早点睡

## 价值主张

唯有早睡不可辜负

## 描述

无奈最近因为各种事情而导致拖延晚于`10:30`睡觉，有时是`mac`或者`win10`上的代码，有时是`ipad`里的剧，有时是`电话`里的女友，所以希望能有个强制性的监督管理机制，强迫自己关机睡觉
